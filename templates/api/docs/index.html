SmartToolPDF API Documentation

Welcome to the SmartToolPDF API! Our API allows you to integrate powerful file conversion capabilities into your applications.

Base URL

https://smarttoolpdf.com/api/v1

Features

  • PDF to DOCX, XLSX, PPTX conversion
  • Document to PDF conversion
  • PDF manipulation (merge, split, compress)
  • Image processing and conversion
  • Video compression
  • Webhook notifications

Authentication

All API requests require authentication using an API key.

Getting Your API Key

  1. Request API access at /api/access/
  2. Wait for approval (1-2 business days)
  3. Receive your API key and secret via email

Using Your API Key

Include your API key in the request header:

Authorization Header 
Authorization: Bearer YOUR_API_KEY

Or use the X-API-Key header:

X-API-Key: YOUR_API_KEY

Quick Start

Get started with the SmartToolPDF API in minutes.

1. Convert PDF to DOCX

cURL 
curl -X POST https://smarttoolpdf.com/api/v1/convert/pdf-to-docx \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@document.pdf"

2. Check Conversion Status

curl https://smarttoolpdf.com/api/v1/conversions/CONVERSION_ID \
  -H "Authorization: Bearer YOUR_API_KEY"

3. Download Result

curl https://smarttoolpdf.com/api/v1/conversions/CONVERSION_ID/download \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -o converted.docx
