How to Use

  1. Desktop: Drag a file onto the upload zone or click "Browse Files"
  2. Mobile: Tap the upload zone or "Browse Files" button
  3. Select a file from your device (PDF, DOC, DOCX, JPG, JPEG, or PNG)
  4. Review the file information displayed
  5. Click "Convert File" to start the upload
  6. Watch the progress bar as your file uploads
  7. Get redirected to the conversion status page
