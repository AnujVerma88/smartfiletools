Getting Started

How to Create an Account

  1. Click the "Sign Up" button in the top right corner
  2. Enter your email address and create a password
  3. Verify your email address by clicking the link sent to your inbox
  4. Start converting files immediately!

Understanding Free vs Premium

Free Account:

  • 10 conversions per day
  • Maximum file size: 10MB
  • All conversion tools available
  • Standard processing speed

Premium Account:

  • Unlimited conversions
  • Maximum file size: 500MB
  • Priority processing
  • Ad-free experience
  • Priority support

File Conversion

How to Convert a File

  1. Select the conversion tool you need (e.g., PDF to DOCX)
  2. Click "Choose File" or drag and drop your file
  3. Wait for the file to upload
  4. Click "Convert" to start the conversion
  5. Download your converted file when ready

Supported File Formats

Document Formats: PDF, DOCX, XLSX, PPTX, TXT

Image Formats: JPG, PNG, GIF, BMP, TIFF, WebP

Video Formats: MP4, AVI, MOV, WMV

File Size Limits

  • Free Users: Up to 10MB per file
  • Premium Users: Up to 500MB per file
  • Enterprise Users: Custom limits available

Account Management

How to Update Your Profile

  1. Log in to your account
  2. Click on your avatar in the top right corner
  3. Select "Profile" from the dropdown menu
  4. Update your information and click "Save Changes"

How to Change Your Password

  1. Go to your Profile page
  2. Click "Change Password"
  3. Enter your current password
  4. Enter and confirm your new password
  5. Click "Update Password"

How to Upgrade to Premium

  1. Visit the Pricing page
  2. Select the Premium plan
  3. Enter your payment information
  4. Confirm your subscription
  5. Start enjoying unlimited conversions!

How to Cancel Your Subscription

  1. Go to your Dashboard
  2. Click "Subscription Settings"
  3. Click "Cancel Subscription"
  4. Confirm cancellation
  5. Your access will continue until the end of your billing period

Troubleshooting

Conversion Failed - What to Do?

If your conversion fails, try these steps:

  1. Check that your file is not corrupted
  2. Verify the file format is supported
  3. Ensure the file size is within limits
  4. Try converting again
  5. If the problem persists, contact support

File Upload Not Working

If you can't upload files:

  • Check your internet connection
  • Try a different browser
  • Clear your browser cache and cookies
  • Disable browser extensions temporarily
  • Try uploading a smaller file first

Download Not Starting

If your download doesn't start:

  • Check your browser's download settings
  • Disable pop-up blockers
  • Try right-clicking and selecting "Save As"
  • Use a different browser

Quality Issues with Converted Files

If your converted file has quality issues:

  • Ensure your original file is high quality
  • Try converting with different settings
  • Check if the source file has the same issues
  • For images, try a different format
  • Contact support for assistance with specific files
