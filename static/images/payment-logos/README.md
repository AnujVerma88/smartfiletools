# Payment App Logos

This folder contains logos for UPI payment apps displayed on the checkout page.

## Required Logo Files:

1. **gpay.png** - Google Pay logo (recommended size: 80x30px)
2. **phonepe.png** - PhonePe logo (recommended size: 80x30px)
3. **paytm.png** - Paytm logo (recommended size: 80x30px)
4. **bhim.png** - BHIM logo (recommended size: 80x30px)
5. **bharatpe.png** - BharatPe logo (recommended size: 80x30px)

## Where to get official logos:

- **Google Pay**: https://pay.google.com/about/brand-guidelines/
- **PhonePe**: https://www.phonepe.com/press/
- **Paytm**: https://paytm.com/press/
- **BHIM**: https://www.npci.org.in/what-we-do/bhim/product-overview
- **BharatPe**: https://bharatpe.com/

## Format Guidelines:
- Use PNG format with transparent background
- Keep consistent height (30px recommended)
- Maintain aspect ratio
- Optimize file size for web

## Usage:
These logos are displayed on the checkout page below the UPI ID to show supported payment methods.
